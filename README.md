
Разработать приложение, которое производит анализ входного текстового файла с использованием переданного модуля. 
Пример запуска готового приложения: java exercise2 input.txt module1.jar 
где 
input.txt – входной текстовый файл 
module1.jar – имя jar файла содержащий модуль анализатора 

Разработать два модуля (на вход модулей подаются строки по очереди из входного файла): 
1. Принимает строку и удаляет все слова через одно 
2. Принимает строку и удаляет двойные пробелы в ней


ReflectionCommon - содержит код, общий для остальных проектов. В частности интерфейс StringConvertable, который реализуют классы модулей.
Данный модуль подключается к остальным трем проектам в качестве внешней библиотеки. 

ReflectionEngine - основной проект. Получает классы из jar-файла, ищет в нем первый класс, который реализует интерфейс StringConvertable
и преобразует каждую строку файла input.txt в соответствии с заданием.

ReflectionFirstModule и ReflectionSecondModule - плагины. Реализуют интерфейс StringConvertable.

Папка jars содержит все скомпилированные файлы.

Запуск: 
	java -jar ReflectionEngine.jar input.txt module1.jar
